## 布局设计

单个 Figure，`make_subplots` 左右分栏：

```
┌──────────────────────────────────────────────────────────────┐
│  资金曲线 & 月度收益明细                                        │
│  ┌────────────────────────┐  ┌──────────────────────────────┐ │
│  │  资金曲线               │  │  月度收益表格                 │ │
│  │  go.Scatter            │  │  go.Table                    │ │
│  │  Y轴: 元，从50万起      │  │  行=每月，列=5列              │ │
│  │  column_width: 60%     │  │  column_width: 40%           │ │
│  └────────────────────────┘  └──────────────────────────────┘ │
└──────────────────────────────────────────────────────────────┘
```

`make_subplots` 参数：
```python
specs=[[{"type": "xy"}, {"type": "table"}]]
column_widths=[0.60, 0.40]
```

## 月度收益表格

### 列定义

| 列名 | 数据来源 | 格式 |
|------|---------|------|
| 年月 | `nav_df` 按月 resample | `2024-03` |
| 月末总资产 | 月末最后一个 `nav` 值 | `523,450 元` |
| 当月盈亏额 | 月末资产 - 月初资产 | `▲ +8,230` / `▼ -3,100` |
| 当月收益率 | `(1+daily_ret).prod()-1` 月内复利 | `▲ +1.60%` / `▼ -0.59%` |
| 累计收益率 | `(月末资产 / INIT_CASH) - 1` | `▲ +4.69%` |

### 颜色规则

- 正值：绿色背景 `#d4edda`，前缀 `▲`
- 负值：红色背景 `#f8d7da`，前缀 `▼`
- 零值：中性灰 `#f5f5f5`，无箭头
- 适用列：当月盈亏额、当月收益率、累计收益率

### 数据计算逻辑

```python
# 月末资产
monthly = nav_df.set_index('date')['nav'].resample('ME').last()

# 当月盈亏额
monthly_pnl = monthly.diff()  # 第一行为 NaN，显示为 "-"

# 当月收益率（复利）
daily_ret = nav_df.set_index('date')['pct_chg']
monthly_ret = daily_ret.resample('ME').apply(lambda x: (1 + x).prod() - 1)

# 累计收益率
cumulative_ret = monthly / INIT_CASH - 1
```
