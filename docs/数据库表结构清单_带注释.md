## 8. others_calendar

**相关任务**: `tushare_others_hktradecal` (港股交易日历), `tushare_others_tradecal` (A股及期货交易日历)
**描述**: 获取交易日历数据，包括港股(HKEX)和A股及中国大陆期货交易所(SSE/SZSE/CFFEX/SHFE/CZCE/DCE/INE)
**主键**: `exchange`, `cal_date`
  | 类别  | 交易所代码 | 说明              |
  |-----|-------|-----------------|
  | A股  | SSE   | 上海证券交易所         |
  | A股  | SZSE  | 深圳证券交易所         |
  | 期货  | CFFEX | 中国金融期货交易所（中金所）  |
  | 期货  | SHFE  | 上海期货交易所（上期所）    |
  | 期货  | CZCE  | 郑州商品交易所（郑商所）    |
  | 期货  | DCE   | 大连商品交易所（大商所）    |
  | 期货  | INE   | 上海国际能源交易中心（上能源） |
  | 港股  | HKEX  | 香港交易所（港交所）      |
  
### 字段定义

| 字段名 | 数据类型 | 约束 | 说明 |
|--------|----------|------|------|
| `exchange` | VARCHAR(10) | NOT NULL | 交易所代码(HKEX港交所/SSE上交所/SZSE深交所/CFFEX中金所/DCE大商所/CZCE郑商所/INE上能源/SHFE上期所) |
| `cal_date` | DATE | NOT NULL | 日历日期 |
| `is_open` | INTEGER |  | 是否交易(0休市/1交易) |
| `pretrade_date` | DATE |  | 上一个交易日 |
| `update_time` | TIMESTAMP | DEFAULT CURRENT_TIMESTAMP | 数据更新时间 |
