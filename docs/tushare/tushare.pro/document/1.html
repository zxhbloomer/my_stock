Tushare Skills 安装指南

在 OpenClaw 生态中，Skills 是一种功能扩展机制。你可以把 Skill 理解为 AI 助手的"超能力包"：

  • 专注领域：每个 Skill 针对特定场景（如天气查询、金融数据、代码开发）
  • 即装即用：安装后 AI 立即获得新能力，无需复杂配置
  • 内置知识：Skill 包含领域知识库和最佳实践
  • 安全可控：明确声明所需权限，运行环境隔离

Tushare Skills 就是这样一个专门为金融数据分析设计的超能力包，让你的 AI 助手能够实时获取股票、基金、期货、宏观经济等各类金融数据。为便于大模型更好的调取数据，请务必安装Tushare skills。 一些基于OpenClaw的最佳实践，可以参阅：如何在OpenClaw（龙虾）里用好Tushare数据 。

安装与准备

步骤1：安装Tushare skills

openclaw里安装

# 使用 clawhub 安装
clawhub install tushare-data

# 升级更新tushare skills
clawhub update tushare-data

也可以直接在openclaw对话框以聊天方式让openclaw自行安装，提示词为：请安装最新版Tushare skills，名称为tushare-data


如果以上方式都不成功，比如线上安装出现类似Error：Rate limit exceeded等问题，可以通过离线文件的方式安装。Tushare Skills下载地址：https://tushare.pro/files/pro/tushare-data.zip ,下载到本地(假如放到“D:\\”目录)后，可以在OpenClaw通过对话框输入提示词：请安装“ D:\\ ”目录下的tushare-data.zip Skills，也可以在类似WorkBuddy平台里直接将这个zip文件导入到技能模块里，即可自动安装成功。


大模型编程环境里安装

# 使用npx命令安装
npx skills add https://github.com/waditu-tushare/skills --skill tushare

步骤2：安装Tushare

# 安装 Tushare（推荐清华镜像加速）
pip install tushare -i https://pypi.tuna.tsinghua.edu.cn/simple

步骤3：获取 Tushare Token

  1. 访问 Tushare Pro 注册账号
  2. 完成信息填写（可免费获得20积分）
  3. 进入个人中心 → 获取 Token

步骤4：配置 Token

  1. 在openclaw里，直接把tushare token给openclaw，让它设置好token，并让其主要保密
  2. 在python环境里，直接运行ts.set_token('你的token')

Tushare skills 最新文件下载及说明

  1. Tushare skills源文件下载：https://github.com/waditu-tushare/skills
  2. 用户可以自行下载Tushare markdown文件制作更丰富和深度的skills
