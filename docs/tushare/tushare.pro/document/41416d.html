比特币每日量价

接口：btc_pricevol
描述：获取比特币历史每日的价格和成交量数据。

输入参数

名称 类型 必选 描述
start_date str Y 开始时间
end_date str Y 结束时间

输出参数

名称 类型 描述
date str 日期
price float 价格
volume float 交易量

接口用法


pro = ts.pro_api()

df = pro.btc_pricevol(start_date='20180101', end_date='20180801')

或者


df = pro.query('btc_pricevol', start_date='20180101', end_date='20180801')
