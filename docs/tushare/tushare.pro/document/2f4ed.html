游资名录

接口：hm_list
描述：获取游资分类名录信息
限量：单次最大1000条数据，目前总量未超过500
积分：5000积分可以调取，积分获取办法请参阅积分获取办法



输入参数

名称 类型 必选 描述
name str N 游资名称


输出参数

名称 类型 默认显示 描述
name str Y 游资名称
desc str Y 说明
orgs None Y 关联机构


接口示例

#代码示例
pro = ts.pro_api()

df = pro.hm_list()
