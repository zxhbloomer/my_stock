# BBI 趋势跟踪策略 v1 — 单股逐一回测（基础版）

## 策略精髓

**一句话：价格穿越 BBI 就买，跌破 BBI 就卖，全仓操作，无任何过滤。**

v1 是最纯粹的 BBI 信号验证版本，用于确认 BBI 穿越信号在 A 股全市场的基础有效性。

---

## 核心算法

### BBI 指标

```
BBI = (MA5 + MA10 + MA20 + MA60) / 4
```

使用 MA(5,10,20,60)，区别于通达信默认的 MA(3,6,12,24)，对中长期趋势更敏感。

### 入场信号（单一条件）

```
前一日收盘 < 前一日 BBI  且  当日收盘 > 当日 BBI
```

价格从下方穿越 BBI（金叉），立即全仓买入。

### 出场信号（单一条件）

```
前一日收盘 > 前一日 BBI  且  当日收盘 < 当日 BBI
```

价格从上方跌破 BBI（死叉），立即全仓卖出。

### 仓位管理

- 全仓买入（可用资金 100%，取整到 100 股）
- 无止损、无止盈、无加仓

| BBI | 本地计算 | 基于前复权收盘价的 MA(5,10,20,60) 均值 |

### 股票筛选

| 条件 | 值 |
|------|----|
| 上市天数 | ≥ 365 天 |
| 流通市值 | ≥ 100 亿元 |
| 日均成交额 | ≥ 5000 万元 |
| 排除 | ST、退市、北交所 |

## 回测框架

使用 **Backtrader** 框架，对全市场每只股票独立回测（多进程并行，`N_WORKERS=8`）。

每只股票初始资金 **10 万元**，独立核算，输出：
- `output/stats_summary.csv` — 全市场统计汇总（胜率、年化、最大回撤、卡玛比）
- `output/trades_detail.csv` — 全部交易明细

## v1 的局限性

v1 验证了 BBI 穿越信号的基础有效性，但存在明显缺陷：
- **无动量确认**：假穿越（震荡市中频繁触发）导致大量无效交易
- **无止损**：单笔亏损无上限
- **无持仓保护**：入场后立即暴露于反向信号
- **全仓操作**：无法分批建仓或加仓

这些问题在 v2 中逐一解决。
