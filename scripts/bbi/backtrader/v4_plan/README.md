# BBI 周度轮动策略 v4_plan — 组合轮动（当前主力版本）

## 策略精髓

**一句话：每周一从全市场 BBI 上方的股票中，选近 5 日涨幅最强的 Top 5，等权持有，下周一换仓。**

v4_plan 彻底抛弃了逐股回测思路，转为**组合轮动**模式，用一个账户持有多只股票，模拟真实实盘操作。

---

## 核心算法

### BBI 指标

```
BBI = (MA5 + MA10 + MA20 + MA60) / 4
```

### 选股逻辑（每周一执行）

**Step 1 — 初筛**：收盘价（前复权）> BBI（前复权）的所有股票

**Step 2 — 排序**：按近 5 日涨幅降序排列

```
ret5 = close[-1] / close[-6] - 1
```

**Step 3 — 持仓**：取 Top N（默认 5 只），等权分配资金

### 换仓规则

- 每周一（交易日中最早一天）执行换仓
- 不在新 Top N 中的持仓：全部卖出
- 新进入 Top N 的股票：按剩余资金等权买入
- 继续持有的股票：不动

### 无复杂出场逻辑

v4_plan **没有** ATR 止损、MACD 死叉、硬止损等出场条件。唯一出场触发：下周一不再入选 Top N。

## 数据来源

| 数据 | 来源表 | 说明 |
|------|--------|------|
| OHLCV + BBI | `tushare_v2.063_stk_factor_pro` | 前复权价格，BBI 本地计算 |
| 资金流向 | `tushare_v2.080_moneyflow` | `lg_net_vol`，**shift(1)** 防泄露 |
| 筹码分布 | `tushare_v2.061_cyq_perf` | `winner_rate`，**shift(1)** 防泄露 |
| 股票基本信息 | `tushare_v2.001_stock_basic` | 名称、上市状态 |

### 防未来数据泄露

盘后数据（资金流向、筹码分布）在 T 日收盘后才可获取，在 `10_prepare_data.py` 中统一做 `shift(1)`，确保 T 日决策只使用 T-1 日数据。

### 股票筛选

| 条件 | 值 |
|------|----|
| 上市天数 | ≥ 365 天 |
| 流通市值 | ≥ 100 亿元 |
| 日均成交额 | ≥ 5000 万元 |
| 排除 | ST、退市、北交所、688 科创板 |

## 回测框架

**纯 Python 实现**，不依赖 Backtrader。单账户持有多只股票，模拟真实组合操作。

输出文件：
- `output/nav_series.csv` — 每日净值序列
- `output/trade_records.csv` — 全部交易明细（含盈亏）
- `output/weekly_records.json` — 每周换仓快照
- `output/last_holdings.json` — 最后一周持仓（供生成下周操作计划）
- `output/report.html` — 完整 HTML 交互式报表

---

## 关键参数

| 参数 | 值 | 说明 |
|------|----|------|
| 初始资金 | 50 万元 | 模拟实盘规模 |
| 持仓数量 | Top 5 | 等权分配 |
| 买入佣金 | 0.05% | |
| 卖出佣金 | 0.15% | 含印花税 |
| 最低佣金 | 5 元 | |
| 无风险利率 | 2% | 用于夏普比率计算 |

---

## 报表内容

`30_generate_report.py` 生成单页滚动 HTML，包含 6 个 section：

1. **净值曲线** — 归一化净值 + 回撤图 + 关键指标表格
2. **资金曲线 & 月度收益** — 绝对金额曲线 + 月度明细表（年份/月份/月末总资产/当月盈亏/当月收益率/累计收益率/年收益率）
3. **年度收益柱状图**
4. **下周操作计划** — 基于最新数据自动生成买卖清单
5. **最近 10 周持仓周报**
6. **历史交易明细**

## 与 v3 的核心差异

| 维度 | v3（逐股回测） | v4_plan（组合轮动） |
|------|--------------|-------------------|
| 回测模式 | 每只股票独立 10 万元 | 单账户 50 万元持有多股 |
| 换仓逻辑 | 无换仓，持有到出场信号 | 每周一强制换仓 |
| 出场条件 | BBI 死叉 + MACD + ATR 止损 | 仅：下周不入选 Top N |
| 选股依据 | 无（全仓持有单股） | 近 5 日涨幅动量排序 |
| 实盘参考性 | 低（无法同时持有 1700 只） | 高（直接输出下周操作清单） |
| 回测框架 | Backtrader | 纯 Python |
