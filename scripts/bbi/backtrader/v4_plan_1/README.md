# BBI 周度轮动策略 v4_plan_1

## 与 v4_plan 的差异

v4_plan_1 在 v4_plan 的基础上，新增了从 v3 移植的完整卖出逻辑。

### 新增卖出条件（按优先级）

| 优先级 | 名称 | 触发条件 | 持仓天数限制 |
|--------|------|----------|-------------|
| 1 | 硬止损 | 亏损 ≥ 8% | 无 |
| 2 | ATR 追踪止损 | 收盘 < 历史峰值 - 4.5×ATR14 | 无 |
| 3 | BBI 死叉 | 前日收盘 > BBI，当日收盘 < BBI | ≥ 20 日 |
| 4 | MACD 死叉 | MACD < Signal 且 MACD < 0 且 BBI 3日下行 | ≥ 20 日 |
| 5 | 筹码胜率过高 | winner_rate（T-1）> 80 | ≥ 20 日 |

原有轮动卖出逻辑（不在 Top N 则周一卖出）保留不变。

### 资金管理

止损卖出后当天不买入，等下周一换仓时资金自动参与分配。

### 新增数据列

`10_prepare_data.py` 在 parquet 中新增：
- `atr14`：Wilder ATR(14)，用于追踪止损
- `macd`：EMA(12) - EMA(26)
- `macd_signal`：MACD 的 EMA(9) 信号线

### 运行方式

```bash
cd scripts/bbi/backtrader/v4_plan_1
python -X utf8 10_prepare_data.py
python -X utf8 20_run_backtest.py
python -X utf8 30_generate_report.py   # 报表在 http://localhost:8085/report.html
```

### 输出文件

与 v4_plan 相同，额外在 `trade_records.csv` 的 `action` 字段中标注止损原因（如"止损卖出(硬止损)"）。

## 回测结果对比（2018-01-08 ~ 2026-04-24，初始资金 50 万）

| 指标 | v4_plan（无止损） | v4_plan_1（含止损） |
|------|-----------------|-------------------|
| 累积净值 | 15.52 | **17.06** |
| 年化收益 | 39.17% | **40.77%** |
| 最大回撤 | -29.04% | **-23.00%** |
| 夏普比率 | 1.590 | **1.836** |
| 卡玛比率 | 1.349 | **1.773** |
| 最终资金 | 776 万 | **853 万** |

止损逻辑在提升收益的同时，将最大回撤从 29% 收窄至 23%，风险调整后收益（夏普/卡玛）均有显著改善。
